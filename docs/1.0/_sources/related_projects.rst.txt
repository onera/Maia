CGNS
----

The CFD General Notation System (CGNS) provides a general, portable, and extensible standard
for the storage and retrieval of computational fluid dynamics (CFD) analysis data.

https://cgns.github.io/

ParaDiGM
--------

The ParaDiGM library provide the developers a progressive framework, which consists of a set of
low-, mid- and high-level services helpfull to write scientific
computing software that rely on a mesh.
